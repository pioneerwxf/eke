安装步骤

  • 第1步，上传

    请选择和论坛编码一致的程序版本，使用 ftp 软件登陆服务器，将程序包 upload 下的全部文件上传到服务器安装路径下，并确保该位置可以被 web 访问到并有执行 php 代码的权限。

  • 第2步，设置目录属性

    如果您的服务器使用 Windows 操作系统，可跳过这一步。

    使用 FTP 软件登录您的服务器，将服务器上以下的目录属性设置为 777

    
	./attachments
	./cache/robot
	./cache/system
	./cache/tpl
	./data
	./data/robot
	./html
	./log

  • 第3步，配置数据库信息

    使用编辑器打开您本地机器上的配置文件（config.php），并相应设置下面的参数：

    
	$dbhost = 'localhost';    这个一般为localhost
	$dbuser = '';             论坛数据库用户名
	$dbpw = '';               论坛数据库密码
	$dbname = '';             论坛数据库名，以上4条空间商都会给出信息，同时，在论坛的config.inc.php里面可以查看到  
	$tablepre = '';           论坛的表名前缀，默认没有修改为cdb_
	$stablepre = '';          SupeSite表名前缀，默认为supe_
	$siteurl = '';            论坛URL地址
	$bbsurl = '';             SupeSite/X-Space访问URL地址
	$dbcharset = '';          此处填写数据库中看到的论坛表的字符集，非常重要。

  • 第4步，执行安装脚本

    您的安装已经进行到最后一步，请在浏览器中运行 install.php，即访问 http://您的域名/SupeSite目录/install.php 以完成最后的安装。

  • 第5步，安装完毕

    您可以使用论坛的用户名密码登陆后台进行管理和发布操作。
