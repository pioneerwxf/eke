Discuz!EXP for Windows

内容简介

    也许您正在考虑使用 Discuz! 架设属于您自己的论坛社区，可 Discuz! 论坛强大的功能和灵活的设置是否又让身为准站长的您担心会驾驭不了 Discuz! 呢？

    Discuz!EXP for Windows 的出现为您解决此顾虑，Discuz!EXP 能够在您本地计算机中简捷快速的搭建一个 Discuz! 论坛环境并且自动安装 Discuz! 论坛。使您能够以论坛管理员身份登录论坛，体验熟悉所有论坛操作，与实际的论坛环境完全一致。

    同时，从 Discuz!EXP 2.0.0 开始，您已经可以使用它作为服务器环境架设实际个人论坛。

Discuz!EXP 特点

    真实 － 与实际论坛完全一样，没有任何功能差异和限制。

    高效 － Discuz!EXP 系列中能够根据用户计算机配置自动优化服务器环境。

    快速 － 您无需担心如何安装配置服务器环境，只需运行双击鼠标运行安装程序，一切就都已搞定。

    小巧 － 与其他 50-60MB 的环境安装包相比，Discuz!EXP 在集成了 Discuz! 论坛程序和数据库后大小仅仅不到 30MB，但效果却丝毫不打折扣。

    干净 － Discuz!EXP 不修改任何系统文件，也不会复制任何文件到系统文件夹，完全独立运行，不会在您的爱机中产生任何“垃圾”。

安装需求及软件版本

  • 系统需求

    操作系统：Microsoft Windows2000/XP/2003

  • Discuz!EXP 集成软件

      • Discuz! 4.x 或者 5.x 简体中文GBK版

      • Discuz! 4.x 或者 5.x 简体中文UTF-8版

      • Discuz! 4.x 或者 5.x 繁体中文BIG5版

      • Discuz! 4.x 或者 5.x 繁体中文UTF-8版

      • Apache 2.0.55-de1

      • MySQL 5.0.18-de2

      • PHP 4.4.2-de1

      • Zend Optimizer 2.6.2-de1

      • phpMyAdmin 2.8.0.1-de1

安装使用

  • 下载 Discuz!EXP：http://www.discuz.net/thread-223885-1-1.html

  • 解压缩 discuzexp2_0_1_xx_xxx.zip 至任意位置

  • 执行 Setup.exe，安装提示进行安装

程序说明

    为了最大程度提高 Discuz!EXP 的兼容性，我们考虑到您的计算机中此时也许正在运行 MySQL，因而 Discuz!EXP 中的 MySQL 使用 6033 端口启动服务以避免与您计算机中运行的 MySQL 产生冲突。

    安装完成后，Discuz! 默认论坛管理员用户名：admin，默认论坛管理员密码：admin。

    FAQ

    • Q：为什么提示 Apache2 启动失败？

      • A：请检查系统中有没有服务器程序（例如 IIS）占用了 80 端口。

    • Q：我的电脑上的IIS已经占用了 80 端口，我该怎么办？

      • A：请先运行“停止服务环境”

      使用记事本等编辑器打开 X:\安装路径\Apache2\conf\httpd.conf 文件

      找到Listen 80，将 80 改为你希望的端口号，例如 8080，保存文件

      运行“启动服务器环境”即可。

    • Q：其他人可以访问我的论坛吗？

      • A：可以，您可以让其他朋友访问您的论坛进行一些测试。

      首先，要确认您的计算机直接与 Internet 相连

      把这个地址告诉您的朋友：http://您计算机的IP地址（例如 http://12.34.56.78）

EasyDiscuz! for Linux

内容简介

    正如您所了解并且可能正在使用的 Discuz!EXP for Windows 一样，EasyDiscuz! for Linux 是一个由 Discuz! 系统环境工程团队历经数月开发的能够在 Linux 服务器平台快速简便优质的搭建 Discuz! 论坛程序运行环境的优质软件。填写一些基本配置即可构建好 Discuz! 论坛。

    EasyDiscuz! for Linux 的诞生使配置服务器环境时的痛苦与茫然成为过去。留给您的只有轻松与便捷，使 Discuz! 论坛离您更近。

系统需求

  • CentOS 或 Red Hat Linux

  • 系统中未安装过 Apache + PHP + MySQL + Zend Optimizer

  • glibc 2.1/2.2/2.3

安装说明

  • 将 EasyDiscuz! 2.5.0L 下载或上传到 Linux 主机，地址：http://www.discuz.net/viewthread.php?tid=220577

  • 进入 EasyDiscuz-2.5.0L 目录，并执行 ./install

  • 出现“Please set MySQL root password:”提示时，请设置 MySQL root 密码

  • 出现“Do you want Apache to start on boot?(y/n)”提示时，询问您是否希望在服务器开机时自动启动 Apache 服务，如果希望请输入 y 回车

  • 出现“Where to create WEB ROOT directory?(default:/home/wwwroot)[DO NOT contain / at the end!!]:”提示时，请输入您想建立网站根目录的路径，例如：/var/www/bbs （★特别注意：输入路径时，路径的末尾不要加/）

  • 出现“Please input your webmaster's e-mail(default:webmaster@mydomain.com):”提示时，请输入您服务器管理员的邮件地址

  • 出现“Please input your server name(default:www.mydomain.com):”提示时，请输入您的服务器域名或主机名

  • 当出现论坛程序版本选择表格时，请输入您想安装的 Discuz! 论坛版本序号。输入数字1-4

  • 出现“Please input BBS Admin e-mail (default: webmaster@mydomain.com):”提示时，请输入 Discuz! 论坛管理邮件地址

使用说明

  • 全部安装完毕后，请打开浏览器输入 http://服务器IP地址，如果能够正常访问 Discuz! 论坛则说明安装成功

  • 默认的论坛管理员帐号为 admin 密码为 admin，请登录论坛后及时更改帐户密码

  • 您可以通过 http://服务器IP地址/phpMyAdmin 进入 MySQL 管理系统

环境说明

  • Apache 安装位置：/usr/local/apache

  • PHP 安装位置：/usr/local/php

  • Zend 安装位置：/usr/local/Zend

  • MySQL root 密码记录文件：/etc/mysql_root_password
